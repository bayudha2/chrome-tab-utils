## Chrome Extension for Tab Utility

feature(for now):

to open command use:
```
Alt + K
```

- collapsing tab group using keyboard shortcut

```
Alt + 1: tab group 1
Alt + 2: tab group 2
Alt + 3: tab group 3
...
Alt + 9: tabg group 9
```

- create new tab to the right (and group it to tab group)
```
Alt + T
```
